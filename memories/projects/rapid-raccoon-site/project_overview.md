Rapid Raccoon is a small indie game studio site at rapid-raccoon.com, deployed to Cloudflare Workers Static Assets (wrangler.toml, custom domain www.rapid-raccoon.com). Astro 6 static site. Tagline: "Small studio. Occasional launches. Mostly self-published."

As of 2026-04-24, three games are on the board, all upcoming placeholders:
- Project Ember — 2026-05-23 launch window (first)
- Project Tide — 2026-06-20 launch window (second)
- Project Nocturne — 2026-08-08 launch window (third)

All three are dev-codename placeholders — real titles/copy/media still to come. The `/products/` content collection is where "launches" live; the URL path stayed `/products/` but the surface label in the nav is "Games".

**Why:** The user wanted a corporate-looking landing + archive site for the indie studio, using Google Stitch's dark cyberpunk "Mascot" home aesthetic. Stitch invented a fictional "urban autonomous logistics" positioning that was explicitly rejected — none of that copy is on the site.

**How to apply:** Treat Rapid Raccoon as an indie game studio, not a logistics/urban-ops/B2B company, regardless of what Stitch assets might suggest. Every "launch" is a game release. Don't reintroduce Stitch's fictional copy (Urban Operating System, Gaze Tracking, telemetry panels, fake delivery stats, carrier login, etc.).
